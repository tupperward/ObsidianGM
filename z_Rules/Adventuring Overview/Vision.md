name: Vision
type: rule
---
Darkness causes those without a Sense capable of otherwise seeing to act as [[Blinded]].

### Senses

| Sense | Description |
|-------|-------------|
| Allsight | Isn't fooled by illusions or invisibility. |
| Blindsight | It can see even if it is [[Blinded]]. |
| Darksight | It can see in darkness. |
| Echolocation | It can see by hearing. |
| Seismicsense | Senses anything touching ground it is touching. |
| Telepathy | It can see anything that can think. |
